# Pharma Drug Sales Analysis
📊 Project Overview
Interactive Power BI dashboard analyzing pharmaceutical drug sales across 4 time-granularity datasets (Daily, Hourly, Weekly, Monthly). Delivers comprehensive insights into sales performance, product trends, temporal patterns, and business optimization opportunities.

#Dataset	Granularity	Key Metrics	Use Case
salesdaily.csv	Daily	Revenue, Units, Products, Regions	Trend analysis, daily performance
saleshourly.csv	Hourly	Revenue, Units by Hour	Peak hour optimization
salesweekly.csv	Weekly	Aggregated sales	Weekly planning & forecasting
salesmonthly.csv	Monthly	Total revenue, growth rates	Strategic reporting
